# HakuLogi - Logistic Cost Estimation Between Indonesia-Japan 🇮🇩🇯🇵🗾

HakuLogi is a specialized platform dedicated for estimating cost when relocating between Indonesia and Japanase. 
HakuLogi offer accurate cost estimation using Yen (¥) and Indonesian Dollar (IDR) 
and extensive options to ensure accurate itemization so you don't have to count individually.

### Core Architecture

HakuLogi use [Laravel 13.x](https://laravel.com/) Extensive Framework as both frontend and backend, 
combined with [Laravel LiveWire 4.x](https://livewire.laravel.com/), [Tailwind v4.2](https://tailwindcss.com/), 
and [Flux UI 2.x](https://fluxui.dev/). HakuLogi is also integrated with [Supabase](https://supabase.com/) as
primary cloud database using PostgreSQL.
