# HakuLogi - Logistic & Items Relocation Between Indonesia-Japan 🇮🇩🇯🇵🗾

HakuLogi is a specialized platform dedicated for estimating cost when relocating between Indonesia and Japanase. 
HakuLogi offer accurate cost estimation using Yen (¥) and Indonesian Dollar (IDR) 
and relocation service to streamline your process on moving from Japan to Indonesia.

### Core Architecture

HakuLogi use [Next JS]([https://laravel.com/](https://nextjs.org/) Extensive Framework as both frontend and backend. 
HakuLogi is also integrated with [Supabase](https://supabase.com/) as primary cloud database using PostgreSQL.
